# Sudoku Solver

## Description
This is a sudoku Solver project developed in C++.

## Features
- Solves a 9*9 Sudoku puzzle
- Uses Backtracking Algorithm
- Simple and easy-to-understand C++ code

## Language
- C++

## How to Run
1. Compile the code.
2. Run the program.
3. Enter the Sudoku puzzle.
4. The solved Sudoku will be displayed.
